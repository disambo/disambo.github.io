在Blender中尽可能将三角面转换为四边面（数学优化）

Blender有一个功能，可以将三角面变成四边面。这会考虑边缘的长度来选择要溶解的边缘。
这本身就很方便，但有时我想尽可能地将三角形脸变成方形脸。在这里，我们介绍一个通过数学优化来实现它的附加组件。

具体例子

尝试下面的网格。

如果执行普通的“三角面转四边面”，则会如下所示。

角上有三角形。

然而，如果运行这次介绍的附加组件，则会如下所示。

所有三角面都完美转变成四边面。

安装方法

安装PULP

该附加组件使用Python 的PuLP模块。
所以你需要在 Blender 中通过命令行安装 PuLP，如下所示。

  • 请选择对应的版本
  • 对于 macOS

/Applications/Blender.app/Contents/Resources/3.1/python/bin/python3.10 -m pip install pulp

  • 对于 Windows

“C:\Program Files\Blender Foundation\Blender 3.1\3.1\python\bin\python” -m pip install pulp
“C:\Program Files\Blender Foundation\Blender 3.1\3.1\python\Scripts\pip” install pulp

  • 在 Windows 上，如果您安装它时出现import错误，请卸载一次然后用管理员权限使用命令提示符重新安装它可能会起作用。实际上，我想使用Python-MIP库，但它不适用于Blender 3.1，所以我使用PuLP。

插件安装

  • 从GitHub上的Tris-Quads-Ex 页面下载ZIP文件。不要解压缩ZIP文件。
    按 Blender 首选项附加组件中的“安装…”按钮，然后选择下载的 ZIP 文件。
    PS：这个插件位置在Blender插件栏的测试项中，所以本身只支持到blender3.4版，因为之后的3.5、3.6已经取消测试项。
  • 这里有在3.5以上版本运行的解决方案，你可以直接下载我修改后的py文件进行安装，在这里下载 。

用法

  • 选择目标对象并进入编辑模式。
    选择感兴趣的区域，然后从面菜单中选择“Tris to Quads Ex”。
  • 它只会在选定的区域进行溶解。如果您想定位全部，请按A选择。但需要注意的是面数越多需要计算的时间越长。
